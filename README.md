


## Backend setup

To setup the booking management system, you will be required to setup all components of the tech stack.

## Tech stack

1. Firebase for NoSql database,
2. Firebase web app hosting for next js frontend and serverside
3. Firebase authentication.
4. Next Js,
5. Stripe payments

## Webpage 

This website is a single page website designed for simplicity and maximise bookings.

### Themes
The website can be both in light and dark theme. The idea is that the dark theme gives the site a cozy aestehtic, similar to a sauna; and the light theme looks sleak and professional.

### Navigation bar
The navigation bar navigates to different elements on the one page. It also allows the user to go from light to dark theme

## Admin page

at /admin, the admin can login using the admin password. There they can add, view and delete bookings. The admin can also set special events.
